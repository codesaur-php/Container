## English

Lightweight, fast, PSR-11 compliant **dependency injection container**.  
This package is a component of the codesaur framework but can be used independently in any PHP project.

### Features

- ‚úî Implements PSR-11 `ContainerInterface`  
- ‚úî Lazy Loading - Services are created only when needed  
- ‚úî Auto-wiring - Automatic dependency resolution  
- ‚úî Interface Binding - Bind interfaces to implementations  
- ‚úî Service Aliases - Access one service by multiple names  
- ‚úî Automatically creates instances from classes using Reflection  
- ‚úî Closure / callable support  
- ‚úî Framework-agnostic - Compatible with all PHP frameworks  
- ‚úî No external dependencies required
